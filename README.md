# Sequential Model Pytorch

The repo provides library supporting:
- RNN, LSTM, GRU, bi-GRU, 1d-CNN, RCNN, etc.
- Adaptable to variable length input sequence.

## Quick start

Choose the type of neural networks in `train.py`

To train:

```angular2html
python train.py
```

To test:

```angular2html
python test.py
```
